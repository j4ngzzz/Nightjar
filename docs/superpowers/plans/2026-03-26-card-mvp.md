# CARD MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `contractd` — a CLI tool and MCP server that parses `.card.md` specs, generates verified code via LLM + Dafny, and provides the verification layer every vibe coding tool needs.

**Architecture:** 5-layer system (spec → generation → verification → output → immune system). Verification pipeline has 5 stages ordered cheapest-first with short-circuit on failure. Retry loop uses Clover pattern [REF-P03] with DafnyPro structured errors [REF-P06]. All LLM calls go through litellm [REF-T16] for model-agnosticism. See `docs/ARCHITECTURE.md` for full design.

**Tech Stack:** Python 3.11+, Click [REF-T17], litellm [REF-T16], Dafny 4.x [REF-T01], Hypothesis [REF-T03], Pydantic v2 [REF-T08], CrossHair [REF-T09], icontract [REF-T10], uv [REF-T05], pip-audit [REF-T06], MCP SDK [REF-T18]

This file defines ALL shared types. Builders import from here. This file is FROZEN after Task 0 — changes require Coordinator approval.

```python
"""Shared types for CARD. All builders import from here.
DO NOT modify without Coordinator approval — this is the shared interface."""

from dataclasses import dataclass, field
from enum import Enum
from typing import Optional

class InvariantTier(str, Enum):
    """[REF-C01] Tiered invariants — CARD's invention."""
    EXAMPLE = "example"    # Unit test only
    PROPERTY = "property"  # Hypothesis PBT auto-generated [REF-T03]
    FORMAL = "formal"      # Dafny mathematical proof [REF-T01]

class VerifyStatus(str, Enum):
    PASS = "pass"
    FAIL = "fail"
    SKIP = "skip"
    TIMEOUT = "timeout"

@dataclass
class Invariant:
    id: str
    tier: InvariantTier
    statement: str
    rationale: str = ""

@dataclass
class ContractInput:
    name: str
    type: str
    constraints: str = ""

@dataclass
class ContractOutput:
    name: str
    type: str
    schema: dict = field(default_factory=dict)

@dataclass
class ModuleBoundary:
    owns: list[str] = field(default_factory=list)
    depends_on: dict[str, str] = field(default_factory=dict)
    excludes: list[str] = field(default_factory=list)

@dataclass
class Contract:
    inputs: list[ContractInput] = field(default_factory=list)
    outputs: list[ContractOutput] = field(default_factory=list)
    errors: list[str] = field(default_factory=list)
    events_emitted: list[str] = field(default_factory=list)

@dataclass
class CardSpec:
    """Parsed .card.md specification."""
    card_version: str
    id: str
    title: str
    status: str
    module: ModuleBoundary
    contract: Contract
    invariants: list[Invariant]
    constraints: dict[str, str] = field(default_factory=dict)
    intent: str = ""
    acceptance_criteria: str = ""
    functional_requirements: str = ""

@dataclass
class StageResult:
    """Result from one verification stage."""
    stage: int
    name: str
    status: VerifyStatus
    duration_ms: int = 0
    errors: list[dict] = field(default_factory=list)
    counterexample: Optional[dict] = None

@dataclass
class VerifyResult:
    """Result from the full verification pipeline."""
    verified: bool
    stages: list[StageResult] = field(default_factory=list)
    total_duration_ms: int = 0
    retry_count: int = 0
```
