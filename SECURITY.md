# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email: nightjar-license@proton.me

Include:
- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

We'll respond within 48 hours and coordinate a disclosure timeline with you.

## Scope

Security issues in Nightjar itself are in scope. Findings from Nightjar's package scanner (bugs in third-party packages) should be reported to those packages directly.

If you use Nightjar to find a security issue in another package and want help with the write-up or disclosure process, we're happy to assist — email us.

## Bug bounty

There's no formal bounty program. If a finding is significant, we'll acknowledge you in the release notes and CHANGELOG.
